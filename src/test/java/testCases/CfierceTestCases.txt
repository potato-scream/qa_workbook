ID 1
Модуль: Поиск товаров
Название: Проверка наличия выдачи результатов по поиску в меню All
Приоритет теста: Высокий
Среда тестирования: PC, Google chrome, версия 126.0.6478.183 (Official Build) (x86_64)
Предварительные условия: Открыт сайт https://www.cfierce.com/
Шаги теста: 1) В меню навигации по сайту кликаем на пункт All. Ожидаемый результат: появляется выпадающий список с поиском по категориям товаров
            2) Выбираем тип товара. Ожидаемый результат: При выборе типа товаров в выдаче есть карточки товаров

ID 2
Модуль: Навигация по сайту
Название: Проверка корректного перехода на страницы Контактов и Блога с главной страницы сайта
Приоритет теста: Высокий
Среда тестирования: PC, Google chrome, версия 126.0.6478.183 (Official Build) (x86_64)
Предварительные условия: Открыт сайт https://www.cfierce.com/
Шаги теста: 1) Нажать кнопку Блог в верхнем меню. Ожидаемый результат: Переходим на страницу блога, текст заголовка страницы News
            2) Нажать кнопку Контакты в верхнем меню. Ожидаемый результат: Переходим на страницу контактов, текст заголовка страницы Contact us
